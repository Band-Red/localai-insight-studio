# Changelog | سجل التغييرات

All notable changes to this project will be documented in this file.

## [6.0.0] - 2026-02-21

### Added

- ميزة التوليد السريع (Speculative Decoding) عبر النماذج المسودة (Draft Models).
- دعم كامل لبروتوكول MCP لإضافة أدوات خارجية.
- المحاكي التفاعلي (Interactive Sandbox) مع المفتش البصري (Visual Inspector).
- لوحة مراقبة متقدمة لـ 8 مقاييس للأداء والنظام.
- التصدير التلقائي لتطبيق Obsidian.

### Fixed

- مشكلة تعليق التبويبات في صفحة النماذج.
- مشكلة إدخال الأرقام العربية في حقول الإعدادات (Context Limit).
- تفعيل زر اختيار المجلدات في الإعدادات بشكل صحيح.
- تصحيح ظهور أكواد المصدر في الـ Previewer.

### Improved

- إضافة مؤشرات النقر (Cursor Pointer) لجميع الأزرار التفاعلية.
- تحسين سرعة جلب إحصائيات النظام عبر الـ Fallback في جسر البايثون.
- دعم كامل لواجهة اللغة العربية (RTL).

## [5.0.0] - 2026-01-15

### Added

- الواجهة الرئيسية بنظام النيون.
- دعم أساسي لنماذج GGUF.
- الدردشة الأساسية.
