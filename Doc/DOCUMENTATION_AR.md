# توثيق المشروع: LocalAI Insight Studio

## 1. مقدمة

**LocalAI Insight Studio** هو بيئة تطوير متكاملة (IDE) للذكاء الاصطناعي المحلي. يركز المشروع على الخصوصية، إنتاجية المطور، والمراقبة الفورية للنماذج. من خلال الابتعاد عن الاعتماد على السحابة، يوفر النظام بيئة مستقرة وآمنة لتطوير البرمجيات المدعومة بالذكاء الاصطناعي.

## 2. معمارية النظام

يعتمد التطبيق على معمارية Electron متعددة العمليات:

- **العملية الرئيسية (Main Process)**: تتولى عمليات مستوى النظام، إدارة الملفات، دورة حياة نموذج الخادم، والاتصال بين العمليات (IPC).
- **عملية العرض (Renderer Process)**: تدير واجهة المستخدم، إدارة الحالة، ومكونات صندوق المعاينة التفاعلي.
- **جسر بايثون (Python Bridge)**: عملية جانبية تجمع مقاييس الأجهزة في الوقت الفعلي (CPU, RAM, GPU) باستخدام مكتبة `psutil`.

## 3. الوحدات الأساسية

### 3.1 الدردشة ونظام RAG

يمكن للمستخدمين الدردشة مع النماذج المحلية وتزويدها بالسياق عبر:

- إرفاق ملفات فردية (.txt, .md, .pdf, .ts، إلخ).
- اختيار مجلدات مشاريع كاملة.
  يقوم النظام بفهرسة هذه الملفات محلياً وتقديمها كـ "سياق" لنموذج الذكاء الاصطناعي.

### 3.2 مدير النماذج (GGUF)

مصمم خصيصاً لنماذج GGUF:

- **المكتبة**: إدارة ملفات النماذج المحلية.
- **إعدادات الخادم**: ضبط المنافذ، تخصيص ذاكرة RAM/VRAM، وإعدادات الأمان.
- **التوليد السريع (Speculative Decoding)**: تهيئة "نموذج مسودة" صغير لتسريع توليد النصوص في النماذج الكبيرة.
- **MCP**: دعم خوادم بروتوكول سياق النموذج للأدوات الخارجية.

### 3.3 صندوق المعاينة والمحاكي التفاعلي

ميزة فريدة تسمح بالمعاينة المباشرة:

- **معاينة الويب**: عرض HTML/JS/CSS في إطار معزول (iframe).
- **المفتش البصري**: حقن كود فحص في المعاينة. عند النقر على عنصر، يمكن توجيه الذكاء الاصطناعي لتعديل ذلك الجزء المحدد فقط من الكود.
- **العرض الخارجي**: خيار بضغطة زر لفتح المعاينة الحالية في متصفح Chrome أو Edge.

### 3.4 لوحة المراقبة (Dashboard)

تعرض صحة النظام وأداء الذكاء الاصطناعي بصرياً:

- جلب البيانات بتردد عالٍ عبر Python.
- رسوم بيانية تفاعلية تعتمد على Recharts.
- مؤشرات حالة لصحة الأجهزة المادية.

## 4. تخزين البيانات

يلتزم المشروع بمبدأ **Local-First**:

- **لا توجد قاعدة بيانات مركزية**: يتم حفظ جميع الحالات عبر ملفات `.json` في المسار `os.homedir()/.localai-insight-studio/`.
- **دعم Obsidian**: يمكن تصدير المحادثات إلى مخازن Markdown القياسية.

## 5. دليل التطوير

### المتطلبات الأساسية

- Node.js (v18+)
- Python 3.x (مع مكتبة `psutil`)
- Git

### التثبيت

```bash
git clone <repository_url>
npm install
pip install psutil
```

### أوامر البناء

- `npm run dev`: يبدأ بيئة التطوير مع مراقبة التغييرات.
- `npm run build`: يولد حزمة الإنتاج باستخدام Webpack.
- `npm run dist`: يحزم التطبيق لنظام Windows بصيغة (.exe).
